En este desafío debes calcular el área de un triángulo, la cuál sigue la siguiente formula:

![area](https://www.voovers.com/wp-content/uploads/2020/05/triangle-area-1-e1629484688356.jpg)

Output

```sh
[4,8,10,12, 16]
```
